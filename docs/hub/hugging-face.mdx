Cortex directly supports all `GGUF`, `ONNX` and `TensorRT` models hosted on [Hugging Face Hub](https://huggingface.co).

## Usage

Simply `cortex pull` the full model path as follows.

```bash 
# cortex pull ORG_ID/MODEL_ID
cortex pull bartowski/Codestral-22B-v0.1-GGUF
```
