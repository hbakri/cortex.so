# `cortex update`

This command updates Cortex to the provided version or the latest version.



## Usage

```bash
cortex update [option]
```
:::info
By default, if no version is specified, Cortex will be updated to the latest version.
:::

## Options

| Option                     | Description                               | Required | Default Value | Example                |
|----------------------------|-------------------------------------------|----------|---------------|------------------------|
| `-v`  | Specify the version of the Cortex.               | No       | -   | `-v 0.5.0`|
