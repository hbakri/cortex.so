# `cortex presets`

This command shows all the available presets within Cortex.



## Usage

```bash 
cortex presets
```

## Options

| Option            | Description                                           | Required | Default value | Example     |
|-------------------|-------------------------------------------------------|----------|---------------|-------------|
| `-h`, `--help`      | Display help information for the command.             | No       | -             | `-h`    |
